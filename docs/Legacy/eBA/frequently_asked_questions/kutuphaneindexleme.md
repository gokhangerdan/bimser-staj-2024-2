# eBA Döküman Yönetimi Kütüphane İndexleme

İlgili kütüphanenin sol tarafında bulunan tiki tıklayın.
Daha sonra üst menüde bulunan Özellikler (Properties) butonuna tıklayın.

![](https://docsbimser.blob.core.windows.net/imagecontainer/ind1-4a4b311c-ac93-407e-81af-2faf79e5c58f.png)

Açılan Properties ekranından Yönetim(Management) sekmesine geçin ve Indexing seçeneğini True olarak seçin ve Apply butonuna basarak ayarları kaydedin.

![](https://docsbimser.blob.core.windows.net/imagecontainer/ind2-da3580c4-7e39-46a5-8746-49eed8bcf41c.png)
