# 'Parolamı Unuttum' Kaldırma

Parolamı unuttum özelliğini kaldırmak için, eBAConfigurationEditor’de Security.PasswordManagement tabındaki ForgottenPasswordEnable değeri false olarak ayarlanır ve kaydedilir.
