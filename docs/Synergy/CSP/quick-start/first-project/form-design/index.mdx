Oluşturulan projenin formunu tasarlamak için; Forms klasörü altındaki varsayılan olarak Form1 adı ile gelen forma çift tıklanır. Ekranın ortasında boş bir form ekranı açılacaktır.

![Form Tasarımı](https://docsbimser.blob.core.windows.net/imagecontainer/auto-upload0c47dd86-5e35-4a5c-8cf1-d7588dac0b23)

Geliştirme arayüzü ekranında Form açıkken, [Araç Kutusu](ide/menu-structure/view/toolbox.md) panelinde, form tasarlarken kullanılabilecek form nesneleri listelenir. Uç kullanıcıların göreceği form yapısı bu nesneler kullanılarak tasarlanır. Araç Kutusu panelinde listelenen nesnelerden tasarım için gerekli olanlar, sürükle/bırak yöntemi ile form üzerinde konumlandırılır.

![Form Tasarımı](https://docsbimser.blob.core.windows.net/imagecontainer/auto-upload5df5bd88-fd5d-469a-aaa0-b5b4eddf7162)

Yıllık İzin Talep sürecinin formunda bulunabilecek başlıca alanlar aşağıdaki gibidir;

| **Form Bilgileri** 	|  	|
|---	|---	|
| Belge Numarası : 	| Forma sistem tarafından verilecek benzersiz id numarasının gösterileceği alandır 	|
| Belge Tarihi : 	| Formun oluşturulma tarihinin gösterildiği alandır 	|
| **Yıllık İzin Talep Eden Kullanıcı Bilgileri** 	|  	|
| Adı Soyadı : 	| Yıllık izin talep eden kullanıcının, yani akışı başlatan kullanıcının isim ve soyisim bilgilerinin gösterileceği alandır 	|
| Departmanı : 	| Akışı başlatan kullanıcının departman bilgisinin gösterileceği alandır 	|
| **İzin Bilgileri** 	|  	|
| İzin Başlangıç Tarihi : 	| İzin başlangıç tarihinin seçileceği alandır 	|
| İşe Dönüş Tarihi : 	| İşe dönüş tarihinin seçileceği alandır 	|
| Vekalet Eden Kullanıcı : 	| İzin süresince izne çıkan kullanıcıya vekalet verecek kullanıcının seçileceği alandır 	|
| İletişim Numarası : 	| İzne çıkacak kullanıcının telefon numarası gireceği alandır 	|
| Açıklama : 	| İzne çıkacak kullanıcı ekstra bir açıklama girmek isterse bu alanda belirtebilir 	|
