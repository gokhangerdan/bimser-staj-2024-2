# Form Üzerinden Akış Üretme

Aşağıdaki kod ile form üzerinden tanımlı bir akış formu açabilirsiniz.

```
this.startAProcess({
                projectName: "Project1",
                flowName: "Flow1",
                typeName: "StartAProcessArgs",
                parameters: [],
                showOn: "drawer",
                size: 2
            });
```
