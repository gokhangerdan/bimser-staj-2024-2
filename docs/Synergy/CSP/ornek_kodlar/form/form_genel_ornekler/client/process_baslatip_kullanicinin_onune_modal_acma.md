# Process Başlatıp Kullanıcın Önüne Modal Açma

Client (ts) Tarafında process başlatıp kullanıcın önüne modal açma ile ilgili  örnek kullanım aşağıdaki gibidir.

	this.startAProcess(
	        {
	        projectName: "HZOTEST",
	        flowName: "Flow2",
	        typeName: "StartAProcessArgs",
	        showOn: "modal"
	        }
	        );
