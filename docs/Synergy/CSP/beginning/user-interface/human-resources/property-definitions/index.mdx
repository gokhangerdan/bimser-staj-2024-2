# Özellik Tanımları

Organizasyonun yapısı gereği kullanıcıların, pozisyonların, ünvanların ve departmanların, standart değerlerinin dışında belirtilmek istenen farklı değerleri de olabilir. Bu tarz standart bilgilerin dışında, kaydedilmek istenen bilgiler, özellik olarak nitelendirilir. Bu özellikler sayesinde organizasyona özgü verilerin saklanması sağlanmış olur.

Arayüzde, “İnsan Kaynakları” bölümü altındaki “Özellik Tanımları” alanından sistemde tanımlı mevcut özelliklere erişilebilir ve yeni özellikler eklenebilir. Özellik Tanımları alanı;

- [Özellikler](./properties.md)
- [Kullanıcı Özellikleri](./user-properties.md)
- [Pozisyon Özellikleri](./position-properties.md)
- [Unvan Özellikleri](./profession-properties.md)
- [Departman Özellikleri](./department-properties.md)
- [Grup Özellikleri](./user-group-properties.md)

alt başlıklarını içermektedir.
